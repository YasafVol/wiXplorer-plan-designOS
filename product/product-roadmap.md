# Product Roadmap

## Sections

### 1. Project Graph
The core layered visualization — an interactive graph of all project entities (Pages, Apps, Tables, Code Files) with color-coded nodes, edges showing connections, layer filters, and a root project node.

### 2. Entity Detail
A side drawer that opens on node click, showing entity-specific metadata sourced from DM, Dev Center, CMS, or Velo — with navigation links to the underlying resource.

### 3. List Explorer
A flat, filterable list/table view of all entities organized by type, for users who prefer tabular browsing over graph exploration.

### 4. Health Monitor
A dashboard of monitoring alerts and errors, each linked to the relevant entity in the graph — with status indicators showing which pages, apps, or code files have active issues.

### 5. Page Notebook
A per-page summary view that narrates what the page does — its apps, data connections, code files, and flows — with optional text-to-speech readout.
